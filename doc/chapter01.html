Introduction:
What is Clouds Analyzer?

Clouds Analyzer is a program that does two things:

  1. Analyze some fractal properties of existing images
  2. Generate some fractals
The two important properties of fractals (images) that the program studies are:
  1. Fractal dimensions
    • Box-counting dimensions
    • Area-perimeter dimensions
    • Rengi dimensions
    • Correlation dimensions
  2. Tecniques of spatial metric
The fractals that the program generate are:
  1. Koch curves
  2. Fractional Brownian movement in 1, 2 and 3D
  3. Hennon atractor
